# Gee Framework Learning

基于 7天实现系列，从零实现 Web、RPC、Cache、ORM 等基础组件。

## 项目结构
```
project/
├── docs/          # 学习笔记和文档
├── geeweb/        # Web 框架实现
├── geerpc/        # RPC 框架实现
├── geecache/      # 分布式缓存实现  
└── geeorm/        # ORM 框架实现
```

## 学习计划

GeeWeb框架（3天）：Day 1-3

* Day 1: HTTP基础、Context设计、Trie树路由
* Day 2: 分组控制、中间件框架、Recovery
* Day 3: 模板引擎、静态文件、错误处理
